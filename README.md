# 🚀 Space Shooter Game

HTML5 Canvas와 JavaScript로 만든 간단한 **우주 슈팅 게임**입니다.  
플레이어는 우주선을 조종해 운석을 피하고 미사일로 제거하며 최대한 오래 생존해야 합니다.

---

## 🎮 게임 특징

- 🚀 화면 아래에서 시작하는 우주선
- ⬅️ ➡️ ⬆️ 방향키 이동
- 🔫 우주선 현재 위치에서 직선으로 발사되는 미사일
- ☄️ 위에서 떨어지는 운석
- ❤️ 목숨 3개
- ⏱ 생존 시간 = 점수
- ⚡ 10초 생존 시 운석 속도 1.5배 증가

---

## 🕹️ 조작 방법

| 키 | 동작 |
|---|---|
| ← | 왼쪽 이동 |
| → | 오른쪽 이동 |
| ↑ | 위로 이동 |
| Space | 미사일 발사 |

---

## 🧠 게임 규칙

- 운석에 닿을 때마다 목숨 1 감소
- 목숨이 0이 되면 게임이 초기화됨
- 미사일이 운석에 맞으면 해당 운석 제거
- 점수는 살아있는 시간(초)과 동일하게 증가

---

## ▶️ 실행 방법

1. 프로젝트 폴더에 `index.html` 파일 생성
2. 게임 코드 붙여넣기
3. 브라우저에서 파일 열기
4. 즉시 플레이 가능 🎉

> 별도의 설치나 라이브러리는 필요하지 않습니다.
